# TsPay Python Client

![Python](https://img.shields.io/badge/python-3.6%252B-blue)
![License](https://img.shields.io/badge/license-MIT-green)
![PyPI Version](https://img.shields.io/pypi/v/tspay-client)

**TsPayClient** is the official Python client library for interacting with the [TSPay.uz](https://tspay.uz) payment gateway API.  
This package provides a simple and convenient way to integrate TSPay payment processing into your Python applications.
